## 2024-05-22 - Accessibility in Visual-Heavy READMEs
**Learning:** Decorative images (separators, aesthetic gifs) in READMEs create noise for screen readers if they lack empty alt attributes. Functional images (badges inside links) need descriptive alt text to make the link meaningful.
**Action:** Always audit images in READMEs: set `alt=""` for decorative ones, and descriptive text for functional ones.
