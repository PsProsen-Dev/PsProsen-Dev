## 2024-05-22 - [Generic Alt Text on Dynamic Images]
**Learning:** Found a pattern of using generic "Typing SVG" or "Project" alt text for highly informative dynamic images (typing animations, project previews). This completely hides the content from screen reader users.
**Action:** When using dynamic image generators (like readme-typing-svg) or project screenshots, always manually override the default `alt` text with the actual content or a descriptive summary.
