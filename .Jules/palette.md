## 2026-01-23 - Decorative Elements in Profile READMEs
**Learning:** High-visual-impact READMEs (Cyberpunk style) use many decorative GIFs and separators (capsule-render, giphy) that default to noisy or missing alt text.
**Action:** Systematically check all purely decorative visual elements and enforce `alt=""` to improve screen reader focus on content.
